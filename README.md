# Fitness_goal_planner
The Fitness Goal Tracker is a Python-based application that helps users calculate their fitness status and plan daily calorie intake to reach a target weight within a specific time.
The project is implemented using both Graphical User Interface (GUI) and file handling, making it user-friendly and practical.

Objectives

Provide an easy-to-use GUI for user input

Calculate BMI and determine BMI category

Calculate BMR based on gender

Estimate daily calorie needs using activity level

Suggest calories required to reach the fitness goal
